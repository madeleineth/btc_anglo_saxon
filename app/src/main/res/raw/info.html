The dictionary data is based on Joseph Bosworth's 1898 "An Anglo-Saxon Dictionary" and the 1921 supplement by Northcote Toller. The scanning project was done by Jason Burton, B. Dan Fairchild, Margaret Hoyt, Grace Mrowicki, Michael O'Keefe, Sarah Hartman, Finlay Logan, Sean Crist, Thomas McFadden, David Harrison, and Sean Crist; that data is in the public domain.

The code for generating the inflected forms and associated data was written by Ondřej Tichý and described in Morphological Analyser of Old English (2017). The code and data are released under the CC BY-NC 4.0 license.

The BT pennant icon was created by David Correll. It is released under the CC BY 4.0 license.
